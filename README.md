# Learning
The main goal of this repository is to track activities that do not require original code/ original thoughts, such as
* working through coding/package tutorials
* implementing examples from math textbooks

## How to check out new branch
git checkout -b [name]

## How to check out existing branch
git checkout [name]

# How to commit to branch
git status
git add -A
git commit -m "Message"
git push origin [name]


# How to update master
git checkout master
git status
git pull origin master
git merge [name]
git push origin master
